**說明：** 門診未完成檢查清單（僅顯示已排定時間檢查，未排定時間不顯示）

## 2. 電子白板 - 查詢維修單

**說明：** 查詢維修單（含財產編號），FLAG<>'6' AND PRO_TYPE='1'（修繕完成已結案不顯示）

| 項目 | 內容 |
|------|------|
| API 名稱 | 電子白板 - 查詢維修單 |
| 測試 URL | `http://oasap1t.kmuh.gov.tw:83/api/ERS?UNITCODE=31304` |
| METHOD | POST |

### 傳入參數

| 參數名稱 | 型態 | 必輸 | 說明 |
|----------|------|------|------|
| UNITCODE | varchar(5) | V | 單位代號 |

### 回傳參數

| 參數名稱 | 型態 | 說明 |
|----------|------|------|
| ERS_NO | varchar(15) | 維修單單號 |
| SER_NO | int | 序號 |
| PRO_NAME | nvarchar(32) | 財產編號（含名稱） |
| NAME | varchar(20) | 維修單位 |
| REP_PERSON | varchar(10) | 維修人員 |
| PRC_TYPE | varchar(4) | 處理狀態分類 |
| EXP_DATE | varchar(10) | 預計完成日期 |
| ACT_DATE | varchar(10) | 實際完成日期 |
| REMARKS | nvarchar(200) | 備註 |
| FLAG | int | 流程狀態：-1=退件、0=申請未送出、2=申請送出主管已簽核、3=處理中、4=修繕完成未驗收、5=修繕完成未結案、6=修繕完成已結案 |

### 工務管理查詢單位請修單（高榮版）

**說明：** 提供該單位 1 個月內提出之已完修及所有未完修請修單資訊

| 項目 | 內容 |
|------|------|
| 正式 URL | `https://eop01p.vghks.gov.tw/ValidateWebService/WebService.aspx?Fun=GetUnitMaintainStatus` |
| METHOD | POST |

#### 傳入參數

| 參數名稱 | 型態 | 必輸 | 說明 |
|----------|------|------|------|
| HID | char(3) | V | 院區編號（3A0） |
| UNITCODE | char(5) | V | 單位代碼（例：73200 = 32病房） |

#### 回傳參數

| 參數名稱 | 說明 |
|----------|------|
| MAINTAIN_NO | 請修單號 |
| EQMT_CODE | 財產碼 |
| EQMT_DESC | 財產名稱 |
| TEAM_DESC | 維修單位 |
| STAFF_NAME | 維修人員 |
| MAINTAIN_IO_TYPE_DESC | 處理狀態（自修、建議報廢、商修、合約） |
| PRE_FINISH_DATE | 預計完成日期 |
| MTN_FINISH_DATE | 實際完成日期 |
| STATUS | 狀態：未分派、未完工、會簽中、未結案、已結案；若無請修單回傳「無請修紀錄」 |
