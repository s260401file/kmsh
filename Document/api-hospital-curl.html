01 未執行檢查及會診清單 高榮 AMDRService
查詢病患未執行的檢查（CHK）、會診（CON）、手術（OP）。僅顯示已排定時間項目。
curl --location 'https://zwmc01t.vghks.gov.tw:4431/AMDRService/amdr/ord/qrynonexschlist' \
  --header 'Content-Type: application/json' \
  --data '{
    "KeyId": "08be7b58-78d6-4ae7-9129-0577e04a4c17",
    "hid":   "4A0",
    "apid":  "APEBOARD",
    "hhisnum": "19000136",
    "hcasetyp": "A"
  }'
期望回傳：{"success":"Y","msg":"","resultList":[...]}　chktype = CHK / CON / OP

05-1 取得急診 / 住院病房清單 高榮 AMDRService
取得可查詢的病房代號清單，E=急診、A=住院。
curl --location 'https://zwmc01t.vghks.gov.tw:4431/AMDRService/amdr/bed/getHnurstaList' \
  --header 'Content-Type: application/json' \
  --data '{
    "KeyId": "08be7b58-78d6-4ae7-9129-0577e04a4c17",
    "hid":    "4A0",
    "hcsetyp": "E",
    "apid":   "APEBOARD"
  }'
期望回傳：{"success":"Y","resultList":[{"hnursta":"MER","hnnsname":"急診室"}]}

curl --location 'https://zwmc01t.vghks.gov.tw:4431/AMDRService/amdr/bed/getBedList' \

    "hnursta":  "MER",

期望回傳：{"success":"Y","hbedList":[{hhisnum, hnamec, hbedno, hcaseno, emgtyp ...}]}
