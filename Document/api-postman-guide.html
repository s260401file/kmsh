架構說明

本專案 ASP.NET Core API（本地端）作為代理層（Proxy），封裝了院方（高榮 VGHKS + 高醫 KMUH）提供的所有 HIS API。

測試時只需呼叫本地 API（http://localhost:5100），不需直接存取院方系統，也不需帶入 KeyId、hid、apid 等認證參數。

來源系統本地 Base URL說明
高榮 VGHKS（AMDRService / UDSPService / LABService）http://localhost:5100透過 IIS ARR 亦可用 http://localhost/api/...
高榮 VGHKS（MAASService）需設定 VghksApi:MaasBaseUrl
高醫 KMUH（HRS / UAS / ERS / TMS / UNIT / CNC）包含 GET + XML 的 CNC 查床號
本地 DB（Whiteboard）http://localhost:5100跑馬燈 / 佈告欄等自建資料，不轉發院方
Postman 設定提醒：在 Postman 中建立 Environment 變數 base_url = http://localhost:5100，各請求使用 {{base_url}}/api/... 即可。
ENDPOINT 01
未執行檢查及會診清單
查詢病患尚未執行的檢查（CHK）、會診（CON）、手術（OP）清單。僅顯示已排定時間的項目。
高榮 AMDRService 本地端點
GET http://localhost:5100/api/NonExSchList?hhisnum=19000136&hcasetyp=A
Query 參數
參數型別必填說明範例值
hhisnumstringY病歷號19000136
hcasetypstringN就醫類型：A=住院、E=急診A
cURL（院方直接測試用）
# 直接呼叫院方測試環境
curl --location 'https://zwmc01t.vghks.gov.tw:4431/AMDRService/amdr/ord/qrynonexschlist' \
--header 'Content-Type: application/json' \
--data '{
  "KeyId": "08be7b58-78d6-4ae7-9129-0577e04a4c17",
  "hid": "4A0",
  "apid": "APEBOARD",
  "hhisnum": "19000136",
  "hcasetyp": "A"
}'
回傳欄位（resultList[]）
欄位說明
chktype類別：CHK=檢查、CON=會診、OP=手術
hnamec病患姓名
orproced檢查/會診項目名稱
orschdt / orschtm排程日期 / 時間
orstatuschn狀態（中文）
hnursta / hbed病房 / 床號

ENDPOINT 05-3
急診病人詳細資料
依病歷號與就醫序號查詢急診病人完整明細（診斷、護理師、DNR、滯留時間等）。
高榮 AMDRService本地端點
GET http://localhost:5100/api/Patient/er?hhisnum=19000136&hcaseno=E20260001
Query 參數
參數必填說明
hhisnumY病歷號
hcasenoY就醫序號（由 05-2 取得）
hnurstaN急診病房代碼（MER/SER/PER）
hbednoN床號
主要回傳欄位（amdrCase）
路徑說明
amdrCase.hpbasic.hnamec姓名
amdrCase.drName / nurseNo負責醫師 / 主責護理師
amdrCase.hemgtype檢傷等級（1–5）
amdrCase.amdays滯留時間（分鐘）
amdrCase.patflag.hicmap.dnrDNR
amdrCase.diagnos.icd10lst診斷碼清單

ENDPOINT 08-1
查床號（KMUH CNC）
依病歷號查詢目前床號、生日、身分證號等基本資訊。院方回傳 XML，本地 API 已轉換為 JSON。
高醫 KMUH CNC本地端點（XML→JSON）
GET http://localhost:5100/api/Patient/cnc?chartNo=19000136
Query 參數
參數必填說明
chartNoY病歷號
回傳欄位（JSON，院方原始為 XML）
欄位說明
bedNo床號（XML: Bed_No）
birthDate生日（XML: Birth_Date）
chartNo病歷號（XML: Chart_No）
idno身分證字號（XML: Idno）
patientName姓名（XML: Patient_Name）
sexId性別（XML: Sex_Id）
查無資料時回傳 404。
ENDPOINT 08-2
查詢病患資訊（MAASService）
依病歷號或身分證字號查詢病患基本資訊（目前病房、床號、生日、性別）。若查無病患回傳 success: N。
高榮 MAASService本地端點
GET http://localhost:5100/api/Patient/info?hhisnum=19000136
Query 參數（擇一填入）
參數必填說明
hhisnum擇一病歷號
hidno擇一身分證字號 / 居留證號
回傳欄位
欄位說明
success / msgY=成功，N=查無病患
hhisnum / hnamec病歷號 / 姓名
hidno身分證字號
hbirthdt.dts出生日期
hsex / hsexc性別代碼 / 中文
hnursta / hbedno病房 / 床號
注意：MAASService 使用不同主機（wac01），需在 appsettings 設定 VghksApi:MaasBaseUrl。
ENDPOINT 09
依標籤檢核急作
輸入檢驗標籤號碼，查詢該標本是否為急作（urgent: Y/N）。
高榮 LABService本地端點
GET http://localhost:5100/api/Lab/urgent?stickrno=1012642002
Query 參數
參數必填說明
stickrnoY標籤號碼
回傳欄位
欄位說明
successY=成功、N=失敗
msg錯誤訊息（成功時為空）
urgentY=急作、N=非急作
回傳範例
{"success": "Y", "msg": "", "urgent": "N"}
