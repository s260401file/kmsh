說明

本規格書由 html_demo/W52-board/ 之 11 個 HTML 頁籤與對應之 js/*Data.js mock JSON 反推產生，用於：

  • 與資訊單位提供之 api-spec.html（HIS / TMS / 院內既有 API）對照確認可介接欄位
  • 盤點需新建之本地 API 端點（不在 HIS API 範圍內者）
  • 作為後續 ASP.NET Core Controller / Service / Model 開發的依據

所有端點預設以 JWT Bearer Token 認證，回應採 { Success, Message, Data } 包裝，PascalCase 欄位對應後端 C# Model Property。

資料來源

  • HIS API：需自 HIS 取得當下病房住院病患（病歷號、姓名、診斷、入院日、主治醫師）

共通回應格式

所有端點皆採用統一回應包裝結構：

欄位型別必填說明
SuccessboolYAPI 呼叫是否成功
MessagestringY失敗時的錯誤訊息（成功時為空字串）
Dataobject—實際資料；失敗時可為 null

HTTP 狀態碼

代碼意義
200成功；Success 可能為 true（資料正常）或 false（業務邏輯錯誤，如查無資料）
401JWT 未提供或已過期
403JWT 有效但無存取此病房權限
404病房代碼不存在
500伺服器錯誤；Success: false 並含 Message

認證

所有端點需於 HTTP Header 帶入：

Authorization: Bearer <JWT_TOKEN>

JWT 有效期 6 小時，到期需重新登入。
