互動規則

  • 總床數 / 住院：僅顯示數字，不可點擊
  • 手術 / 檢查 / 會診：點擊後過濾含對應標記的床位
  • 隔離：點擊後過濾所有隔離床位（接觸 / 飛沫 / 空氣）
  • DNR：點擊等同 filter bar 點擊「DNR」病人註記，兩者同步高亮
  • RRT：點擊等同 filter bar 點擊「RRT」病人註記，兩者同步高亮
  • 人工血管 / 雙腔靜脈 / 導尿管 / 中心靜脈 / 心導管：點擊後過濾具有該管路的床位（§7.1.2.1 (2.6)）
  • 再點一次：取消過濾，回到全部顯示
六、圖例 / 過濾列操作
圖例 / 過濾： DNR 高危跌 依賴L1 依賴L2 依賴L3 隔離 保密 禁治療 禁食 過敏 RRT 化療 輪椅 推床 氧氣設備 洗腎 待轉入 待轉出 待出院

操作方式

  • 19 個彩色按鈕同時作為圖例與過濾器（13 種病人註記 ＋ 3 種床位動態 + 依賴 L1/L2/L3 細分）
  • 點擊任一按鈕 → 僅顯示具有該標記的床位（空床不受影響）
  • 被選中按鈕出現白色光暈邊框；DNR 統計欄 / DNR 按鈕、RRT 統計欄 / RRT 按鈕互相聯動
  • 再點一次同一按鈕 → 取消過濾，恢復全部顯示
  • 點擊「全部」→ 強制還原所有床位顯示
