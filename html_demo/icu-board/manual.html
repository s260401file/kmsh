3.3 點擊卡片

開啟病人詳細資料 Modal

  • 點擊任何非空床的床位卡片，彈出詳細資料視窗
  • 顯示：床號、姓名、性別/年齡、診斷、病歷號、生日、科別
  • 顯示：主治醫師、責任護理師、入院日期、住院天數、病況等級
  • 顯示：隔離狀態、DNR、呼吸器、CRRT、管路、備註
  • 點擊視窗外部或「關閉」按鈕可關閉

互動規則

  • 總床數 / 住院：僅顯示數字，不可點擊過濾
  • 手術 / 檢查 / 會診 / 穩定 / 重症 / 危急 / 隔離：點擊後過濾符合條件的床位，其餘淡出
  • DNR：點擊等同 filter bar 點擊「DNR」病人註記，兩者同步高亮
  • RRT：點擊等同 filter bar 點擊「RRT」病人註記，兩者同步高亮
  • 氣管內管 / 鼻胃管 / 導尿管 / 中心靜脈：點擊後過濾具有該管路的床位（§7.1.2.2 (1.5)）
  • 再點一次：取消過濾，回到全部顯示（無須移動至左上「全部」按鈕）
六、圖例 / 過濾列操作
圖例 / 過濾： DNR 高危跌 依賴L1 依賴L2 依賴L3 隔離 保密 禁治療 禁食 過敏 RRT 化療 輪椅 推床 氧氣設備 洗腎

操作方式

  • 16 個彩色按鈕（13 種規格 病人註記 ＋ 依賴 L1/L2/L3 細分）同時作為圖例與過濾器
  • 點擊任一 病人註記 按鈕 → 僅顯示具有該標記的床位（空床不受影響）
  • 被選中的按鈕出現深色光暈邊框；DNR 統計欄 / DNR 按鈕、RRT 統計欄 / RRT 按鈕互相連動
  • 再點一次同一按鈕 → 取消過濾，回到全部顯示
  • 點擊「全部」→ 強制還原所有床位顯示
