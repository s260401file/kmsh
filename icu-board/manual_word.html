三、病床卡片說明

3.1 狀態顏色

左側邊框顏色卡片狀態說明
綠色 occupied（住院中） 一般住院病人
紅色 isolation（隔離） 需執行接觸 / 飛沫 / 空氣隔離措施
琥珀 transfer（待轉） 預計當日轉至其他病房或院外
淺綠 discharge（待出院） 預計當日辦理出院
虛線 empty（空床） 無病人，不可點擊

3.2 卡片欄位結構

列別內容
第一列床號（樓層-編號，例：4F-05）
第二列病人姓名（性別顏色，男=藍、女=粉紅） ＋ 性別/年齡
第三列病人註記 badges（最多顯示 3 個，超出以 +N 標示）

3.3 點擊卡片 — 詳細資料 Modal

Modal 顯示欄位清單

  • 床號、姓名、性別 / 年齡、所有 病人註記 badges
  • 診斷
  • 病歷號、生日、科別
  • 主治醫師、責任護理師
  • 入院日期、住院天數、病況等級
  • 隔離狀態、DNR
  • 呼吸器、CRRT
  • 管路（氣管內管 / 鼻胃管 / 導尿管 / 中心靜脈導管，無則顯示「無」）
  • 備註

點擊視窗外部或「關閉」按鈕可關閉 Modal。

五、統計面板操作

統計面板位於主畫面右下方（3F 床位區旁），分為三列：

第一列：床位流量

項目色彩說明可點擊過濾
總床數—ICU 總床位數（4F+3F）否
住院—目前在床病人數否
手術 ▾深藍當日預約手術人數是
檢查 ▾深橘當日預約檢查人數是
會診 ▾深橘紅當日會診人數是

第二列：病況 & 特殊註記

項目色彩說明可點擊過濾
穩定（A）▾深綠病況穩定人數是
重症（B）▾深琥珀重症人數是
危急（C）▾深紅危急人數是
隔離 ▾深粉紅隔離中床數是
DNR ▾深紫DNR 人數；點擊與 filter bar DNR 按鈕連動是
RRT ▾深紅RRT 啟動人數；點擊與 filter bar RRT 按鈕連動是

第三列：管路統計（§7.1.2.2 (1.5)）

項目色彩對應欄位可點擊過濾
氣管內管 ▾深橘ventilator = true是
鼻胃管 ▾深紫ng = true是
導尿管 ▾深青foley = true是
中心靜脈 ▾深綠cvc = true是

互動規則

  • 總床數 / 住院：僅顯示數字，不可點擊過濾
  • 手術 / 檢查 / 會診 / 穩定 / 重症 / 危急 / 隔離：點擊後過濾符合條件的床位，其餘淡出
  • DNR：點擊等同 filter bar 點擊「DNR」病人註記，兩者同步高亮
  • RRT：點擊等同 filter bar 點擊「RRT」病人註記，兩者同步高亮
  • 氣管內管 / 鼻胃管 / 導尿管 / 中心靜脈：點擊後過濾具有該管路的床位
  • 再點一次：取消過濾，回到全部顯示

操作方式

  • 點擊任一 病人註記 按鈕 → 僅顯示具有該標記的床位（空床不受影響）
  • 被選中的按鈕出現深色光暈邊框；DNR 統計欄 / DNR 按鈕、RRT 統計欄 / RRT 按鈕互相連動
  • 再點一次同一按鈕 → 取消過濾，回到全部顯示
  • 點擊「全部」→ 強制還原所有床位顯示
